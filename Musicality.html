Musicality

There are four Musicality tools, selected by the buttons above:

  • Recognise - Name the interval or chord type
  • Sing - Sing the note at an interval from another note or chord
  • Sequence - Sing a sequence of notes in a chosen key
  • Transcribe - Transcribe a sequence of notes in a key

Each tool has three buttons:

  • New - Start a new exercise with a new randomised selection of notes, based on any options selcted
  • Repeat - Play the start notes again
  • Answer - Show and/or play the answer to the current exercise
